# QR Kod Oluşturucu (Python)

Bu proje, Python kullanılarak geliştirilmiş basit bir **masaüstü QR Kod Oluşturucu** uygulamasıdır.  
Kullanıcıdan alınan URL bilgisini QR koda dönüştürür ve **SVG formatında** bilgisayara kaydetmeyi sağlar.

## 🧩 Özellikler

- Kullanıcı dostu grafik arayüz (Tkinter)
- Girilen URL’den QR kod oluşturma
- QR kodu SVG formatında kaydetme
- Dosya kaydetme konumu kullanıcı tarafından seçilebilir
- Basit ve anlaşılır kod yapısı

## 🛠 Kullanılan Teknolojiler

- **Python 3**
- **Tkinter** – Grafik arayüz
- **pyqrcode** – QR kod oluşturma
- **tkinter.filedialog** – Dosya kaydetme penceresi

## 🚀 Kurulum ve Çalıştırma

### 1️⃣ Gerekli Kütüphaneleri Yükleyin

```bash
pip install pyqrcode pypng
Not: pypng, QR kodun düzgün oluşturulabilmesi için gereklidir.
python QR_olusturucu.py
```
## 🖥️ Uygulama Kullanımı

Programı çalıştırın

Açılan pencerede URL bilgisini girin

“QR Kodu Oluştur” butonuna tıklayın

QR kodun kaydedileceği yeri seçin

QR kodunuz SVG formatında oluşturulsun 🎉
